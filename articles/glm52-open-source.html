GLM 5.2 正式开源！百万上下文、Agent 编程能力登顶，附本地部署教程

智谱 AI 正式发布并开源了全新的旗舰模型——GLM 5.2。这是继 GPT-5 之后最受关注的开源大模型发布。与以往单纯提升聊天能力不同，GLM 5.2 的重点放在了 Agent（智能体）与长周期任务处理能力。

更重要的是，GLM 5.2 采用 MIT 开源许可证，允许个人和企业自由使用、修改和商业部署，没有地域限制。

🔥 GLM 5.2 重大升级亮点

1、首次实现稳定的百万 Token 上下文

目前不少大模型都声称支持长上下文，但实际使用中往往会出现"上下文丢失"问题。GLM 5.2 采用了全新的注意力机制优化，在 100 万 token 上下文窗口内，可以稳定完成以下任务：

  • 一次性阅读并总结整本技术书籍
  • 分析跨越数百个文件的代码库架构
  • 处理超长对话历史而不遗忘早期关键信息

2、Agent 编程能力大幅提升

GLM 5.2 在编程相关任务上进行了专项优化：

  • 代码生成：支持 30+ 编程语言，代码质量接近 GPT-4 水平
  • 代码修复：能够理解 Bug 上下文，给出精准修复建议
  • 项目级任务：可以自主分解复杂项目需求，分步骤完成开发
  • 测试生成：自动生成单元测试，覆盖率可达 85% 以上

3、多模态能力增强

GLM 5.2 的多模态理解能力也有显著提升：

  • 支持图片、图表、流程图的深度理解
  • 可以描述视频内容并回答相关问题
  • 对截图、UI 设计稿有准确的分析能力

💻 本地部署教程

方案一：Ollama 部署（推荐新手）

# 安装 Ollama（macOS/Linux/Windows）
# 下载地址: https://ollama.com/download

# 下载 GLM 5.2 模型（约 10GB）
ollama pull glm5:7b

# 启动服务
ollama serve

# 测试运行
ollama run glm5:7b "你好，介绍一下GLM 5.2"

方案二：LM Studio（桌面应用）

LM Studio 提供图形化界面，适合不习惯命令行的用户：

  1. 下载 LM Studio：https://lmstudio.ai
  2. 搜索"GLM-5.2"并下载模型
  3. 在本地聊天界面直接使用

方案三：vLLM 高性能部署

# 克隆 GLM 仓库
git clone https://github.com/THUDM/GLM-5.git
cd GLM-5

# 安装依赖
pip install -r requirements.txt

# 启动推理服务
python serve.py --model-name glm5-7b --port 8000
